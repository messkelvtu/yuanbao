# B站音乐提取器

一个基于Python的桌面应用程序，可以从哔哩哔哩视频中提取音乐并进行管理。

## 功能特点

- 从B站视频链接提取音频（支持单曲和批量下载）
- 自动匹配和搜索歌词
- 音乐文件管理（重命名、分类、移动等）
- 批量操作支持

## 下载安装

从 [Releases页面](https://github.com/yourusername/bilibili-music-extractor/releases) 下载最新版本的 `BilibiliMusicExtractor.exe`。

## 使用说明

1. 启动程序
2. 在输入框粘贴B站视频链接
3. 选择单曲下载或批量下载
4. 下载完成后可在音乐库中管理文件
